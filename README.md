# 19fetch

## 19fetch displays 

* The logo
* The current time
* Your uptime


![19fetch in action](https://github.com/Algotron/19fetch/blob/master/19fetch.png)

# Installation
* clone this repository anywhere you want :^)
* add the *following lines* to `.zshrc`
```
clear
/path/to/19fetch
```
* you're good to go
